# PeachyPet i18n 国际化实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为 PeachyPet 的功能型 UI 引入中英文双语，设置页新增 Appearance section 可实时切换语言，无需重启。

**Architecture:** `@Observable LanguageManager` 单例持有当前语言和覆盖 `Bundle`；全局 `t(_ key:)` 函数取本地化字符串；SwiftUI 通过 `@Observable` 自动追踪依赖，切换时重渲染。

**Tech Stack:** Swift 5.9+, SwiftUI, `@Observable`, `Bundle(path:)` 覆盖，标准 `.lproj/Localizable.strings`。

## Global Constraints

- macOS 14+ 部署目标
- 品牌词不翻译：PeachyPet、Claude Code、Codex、Ghostty、Cursor、iTerm2、VS Code 等
- `t()` 的 fallback 是 key 本身，不会崩溃
- 不改：`OnboardingView`、`PeachyDashboardView`、`MascotDetailView`
- `swift build` 必须在每个 commit 后通过

- [ ] **Step 3: 创建 `LanguageManager.swift`**

```swift
// Sources/Utilities/LanguageManager.swift
import Foundation
import Observation

enum AppLanguage: String, CaseIterable, Identifiable {
    case en = "en"
    case zh = "zh"

    var id: String { rawValue }

    var displayName: String {
        switch self {
        case .en: return "English"
        case .zh: return "中文"
        }
    }
}

@Observable
final class LanguageManager {
    static let shared = LanguageManager()

    private(set) var language: AppLanguage = .en
    private(set) var bundle: Bundle = .main

    private init() {
        let stored = UserDefaults.standard.string(forKey: "appLanguage") ?? "en"
        let lang = AppLanguage(rawValue: stored) ?? .en
        setLanguage(lang)
    }

    func setLanguage(_ lang: AppLanguage) {
        language = lang
        UserDefaults.standard.set(lang.rawValue, forKey: "appLanguage")
        if let path = Bundle.main.path(forResource: lang.rawValue, ofType: "lproj"),
           let langBundle = Bundle(path: path) {
            bundle = langBundle
        } else {
            bundle = .main
        }
    }
}

func t(_ key: String) -> String {
    LanguageManager.shared.bundle.localizedString(forKey: key, value: key, table: nil)
}
```
